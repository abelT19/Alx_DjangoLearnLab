# LibraryProject

This is a basic Django project created to learn Django’s project structure and setup process.

## Steps to Run
1. Install Django using `pip install django`
2. Create the project using `django-admin startproject LibraryProject`
3. Navigate to the project folder: `cd LibraryProject`
4. Run the server: `python manage.py runserver`
5. Open `http://127.0.0.1:8000/` in your browser

## Files
- **manage.py** — Command-line utility to manage the project.
- **settings.py** — Configuration for the Django project.
- **urls.py** — URL routing for the project.
